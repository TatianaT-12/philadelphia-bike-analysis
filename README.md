This repository contains an exploratory data analysis of Indego bike share trips in Philadelphia during the first quarter of 2021. The project focuses on cleaning and analyzing trip-level data to better understand rider behavior and usage patterns.

## Project Overview
Using R, I analyzed bike share trips to explore:
- Daily ride trends by trip route category (one-way vs. round-trip)
- Trips with durations greater than 60 minutes
- The top 10 most frequently used bikes in Q1 2021

The goal of this project is to demonstrate data cleaning, transformation, and visualization skills commonly used in business analytics.

## Data
The dataset includes trip duration, start time, start location, bike ID, and trip route category.  
Source: Indego Bike Share (Q1 2021 data used for academic analysis).

## Methods & Tools
- Data cleaning and manipulation: `dplyr`
- Data visualization: `ggplot2`
- Programming language: R

## How to Use This Repository
1. Clone or download the repository
2. Open the R scripts in the `R/` folder
3. Run the scripts to reproduce the analyses and visualizations
